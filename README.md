## 介紹
此程式協助你查看每個科目的成績分項詳細數據
## 使用方法

1. [下載 msi 安裝檔](https://github.com/MO7YW4NG/CYCU-iLearning-Score-Analyzer/releases)
2. 安裝後生成 `CYCU-iLearning-Score-Analyzer` 資料夾
3. 開啟 `app.exe` 後登入即啟用!
